# .agents/ Directory

Agent skills and reference materials used by automation for this repository.

**Project:** WasmBoy-Voxel — a WasmBoy fork with PPU snapshot APIs for voxel
rendering. AssemblyScript core, JS wrapper + workers, TS voxel wrapper.

## Directory Structure

```
.agents/
└── skills/              # Skill definitions (SKILL.md per skill)
```

Use skills that match the current task and avoid ones that assume unrelated
stacks.
